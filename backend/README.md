### Deck 테이블 삭제 (8/27)
- 한 명의 유저(User)가 여러 학습 목록(Deck)을 가질 수 있도록 구현하려던 방식에서, 하나의 학습 목록만 가질 수 있도록 변경합니다.
- 기존의 Deck 테이블을 삭제하고, reviewCard 테이블의 deck_id를 user_id로 변경합니다.
- 프로젝트 구현 목적에 보다 적합하고, 간단한 구조로 변경합니다.
